전체 채용시장 vs 마케팅 채용공고

파란선 — 전체 공고 수 / 코랄선 — 마케팅 공고 수 (모집단 절대치)
데이터 출처데모 목업 데이터 (실제 서비스에서는 워크넷 공공 채용정보 API + 사용자 제보 통합)
집계 방식각 기간 단위로 마감되지 않은 신규 등록 공고를 합산. 동일 회사 동일 직무는 1건으로 중복 제거
마케팅 필터링 근거공고 직무 카테고리(NCS 분류 코드 02 영업·마케팅 중 02-마케팅 세부) + 제목·본문에 마케팅 관련 키워드(마케팅, 마케터, 브랜드, 콘텐츠, 광고기획, 미디어, CRM, 그로스 등) 포함 시 마케팅으로 분류

전체 대비 마케팅 비율

전체 공고 중 마케팅 직무가 차지하는 비율
계산식(마케팅 공고 수) ÷ (전체 공고 수) × 100. 각 데이터 포인트는 절대치도 함께 표시 (그래프 위 숫자)
데이터 출처위 그래프와 동일

마케팅 신입 채용 비율

마케팅 직무 공고 중 신입 채용 비율 — 코랄선 / 전체 시장 신입 비율 — 회색선
신입 판정 기준공고 경력란이 ① 신입 ② 신입·경력 무관 ③ 경력 0~1년 중 하나에 해당하면 신입 공고로 분류 (워크넷 API 경력 코드 + 텍스트 패턴 매칭)
계산식(신입 가능 마케팅 공고) ÷ (전체 마케팅 공고) × 100, (신입 가능 전체 공고) ÷ (전체 공고) × 100

마케팅 세부 직무별 채용공고 수

기간 동안 직무별 누적 공고 수 — 절대치
세부 직무 분류 근거공고 제목·본문의 키워드 매칭 — 콘텐츠(콘텐츠, 매거진, 에디터, SNS), 퍼포먼스(퍼포먼스, 광고운영, 메타광고, 구글애즈, ROAS), 브랜드(브랜드, BX, 브랜딩), 그로스(그로스, 프로덕트 마케팅, A/B 테스트, 퍼널), CRM(CRM, 리텐션, Braze, 푸시·이메일), AE(AE, 광고기획, 어카운트), MP(미디어플래너, 미디어 플래닝), CW(카피라이터, CW, 카피)
중복 처리한 공고가 여러 키워드에 매칭될 경우, 가중치 점수가 가장 높은 1개 직무로만 집계 (대표 직무 우선순위 적용)
탭 구분전체 공고는 신입·경력 모두 포함, 신입 공고만은 ① 신입 ② 신입·경력 무관 ③ 경력 0~1년에 해당하는 공고만 카운트

직무별 점유율

마케팅 공고 중 각 직무가 차지하는 비중
계산식각 직무의 점유율(%) = (해당 직무 공고 수) ÷ (전체 마케팅 공고 수) × 100. 막대 위 숫자는 절대치(공고 수), 막대 옆 숫자는 비율(%)

직무별 신입 채용 비율

각 직무 공고 중 신입을 받는 비율 — 절대치(N건/N건)와 비율(%)을 함께 표시
계산식(직무별 신입 공고 수) ÷ (직무별 전체 공고 수) × 100. 우측 절대치는 신입 공고 수 / 전체 공고 수 형태로 표기
