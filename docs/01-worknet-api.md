## 승인 떨어지면 어떻게 추가하나요?

### ① 인증키 확인
1. 고용24 마이페이지 → 데이터활용 → 개발계정
2. 신청한 API 클릭 → **일반 인증키 (Decoding)** 복사

### ② Vercel 환경변수에 추가
1. Vercel 대시보드 → 프로젝트 → Settings → Environment Variables
2. **+ Add** 클릭
3. Name: `WORKNET_API_KEY`, Value: 복사한 키 붙여넣기
4. **Save**
